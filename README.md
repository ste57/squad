<h3 align="center"><b>squad</b></h3>

<p align="center">
  A role system for AI agents.<br>
  One role, every project.
</p>

<h4>Get Started</h4>

Paste into Claude Code:

```
Clone https://github.com/ste57/squad.git to ~/.squad and install the skill from ~/.squad/skill, then run /squad
```

<h4>Custom Roles</h4>

Describe what you need. Squad figures out the type, names it, wires it up.

```
/squad create
/squad edit
/squad delete
```

<sub>"I need someone who reviews security for the engineer role" → <code>/squad engineer/security</code></sub>
