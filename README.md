<h3 align="center"><b>squad</b></h3>

<p align="center">
  A role system for AI agents.<br>
  One role, every project.
</p>

<h4>Paste into Claude Code · Restart · /squad</h4>

```
Install https://github.com/ste57/squad.git to ~/.squad as a skill
```

<h4>Custom Roles</h4>

```
/squad create      "make me an accessibility auditor"
```

```
/squad update      "make the a11y auditor check for color contrast too"
```

```
/squad delete      "remove the a11y role"
```
